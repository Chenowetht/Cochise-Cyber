01. NETWORK AUTH

Centralizing authentication for enterprise devices.

RADIUS

Remote Authentication Dial-In User Service

A client/server protocol that provides centralized AAA for users who connect and use a network service.

  • UDP-Based: Fast but connectionless.
  • Encryption: Only encrypts the password.
  • Use Case: VPNs, 802.1X wireless, and dial-up.

"The standard for network access control."

TACACS+

"Designed for administrator access to network hardware."

Terminal Access Controller Access-Control System Plus

A Cisco-proprietary (but widely supported) alternative to RADIUS.

  • TCP-Based: Reliable connection.
  • Encryption: Encrypts the entire payload.
  • Separation: Separates Authentication, Authorization, and Accounting functions.

Kerberos

Ticket-Based Auth

The default authentication protocol for Microsoft Active Directory. Prevents cleartext password transmission.

  • Uses Tickets for access.
  • Requires precise Time Sync (NTP).
  • Features the KDC (Key Distribution Center).

"Single Sign-On for internal networks."

02. WEB & FEDERATED AUTH

Modern standards for cloud and browser-based access.

SAML 2.0

XML Assertions

Exchanges info between an IdP (Identity Provider) and an SP (Service Provider).

Web SSO

Security Assertion Markup Language. Used extensively for logging into cloud apps (like Salesforce or G-Suite) via a corporate login page.

  • Prevents the need to store passwords in multiple cloud apps.

OAuth vs. OIDC

OAuth 2.0 (Authorization)

An open standard for token-based authorization. It grants apps access to resources without sharing credentials.

"Log in with Google to post to Twitter."

OIDC (Identity)

OpenID Connect. An identity layer built ON TOP of OAuth 2.0. It provides the actual "Identity" (Who are you?).

EAP & 802.1X

  • EAP (Extensible Authentication Protocol): A framework that allows different auth methods (Smart cards, certificates, etc.).
  • EAP-TLS: Highest security. Requires certificates on both server and client.
  • 802.1X: Port-based Network Access Control (PNAC). Uses EAP to authenticate devices before granting network access.
