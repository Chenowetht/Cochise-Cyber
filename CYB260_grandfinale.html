ROUND 1

The Handshake

Establish your "Security Score" with the fundamentals.

Assessment 2.1

Which AWS service could an administrator use to securely control access to AWS resources for users?

  • A. AWS Config
  • B. AWS Identity and Access Management (IAM)
  • C. AWS CloudTrail
  • D. AWS CloudFormation

Correct Answer: B. AWS IAM

The "Front Gate" Logic: AWS provides physical security for the data centers. Who is responsible for this? The Landlord (AWS). Principle: Security OF the Cloud.

Assessment 2.2

What are TWO advantages of cloud computing? (Select TWO)

  • A. Provision resources up front
  • B. Stop guessing capacity
  • C. Trade fixed expense for variable expense
  • D. Trade variable expense for fixed expense

Correct Answer: B & C

The "Open Window" Dilemma: A tenant leaves their RDP/SSH ports open to 0.0.0.0/0. Who is at fault? The Tenant (Customer). Security IN the cloud.

Assessment 2.3

What is the triad originally developed to highlight important aspects of information security?

  • A. Visibility, agility, and automation (VAA)
  • B. Confidentiality, integrity, and agility (CIA)
  • C. Confidentiality, integrity, and availability (CIA)
  • D. Controllability, auditability, and agility (CAA)

Correct Answer: C. Confidentiality, integrity, and availability (CIA)

The "Master Key" Service: Which service acts as the "Key Office" to create specific keys for workers? AWS IAM.

Assessment 2.4

Which security principle involves monitoring, alerting, and auditing actions and changes to the environment in real time?

  • A. Enable traceability
  • B. Secure all layers
  • C. Apply the principle of least privilege
  • D. Protect data in transit and at rest

Correct Answer: A. Enable traceability

The "Building Blueprint": If you move furniture to a different complex in Germany to reduce lag, what are you choosing? An AWS Region.

Assessment 2.5

What is a characteristic of the security principle to protect data in transit?

  • A. Use a defense in depth approach
  • B. Use Transport Layer Security (TLS) connections
  • C. Enforce separation of duties
  • D. Be ready to scale and absorb the attack

Correct Answer: B. Use Transport Layer Security (TLS) connections

The "Maintenance Log": The Landlord records every time a key is used. What is this service called? AWS CloudTrail.

Assessment 2.6

What are TWO characteristics of the principle to "prepare for security events"? (Select TWO)

  • A. Use subnets in only one Availability Zone
  • B. Routinely practice incident response through game days
  • C. Create processes to isolate incidents and restore operations
  • D. Use different AWS services
  • E. Use hard-coded credentials

Correct Answer: B & C

The "Shared Utility": Who is responsible for patching the "pipes" (Hypervisor) vs the "sink" (EC2 OS)? AWS patches pipes; Customer patches sink.

Assessment 2.7

Which action is the customer's responsibility under the AWS shared responsibility model?

  • A. Patching the underlying infrastructure
  • B. Securing the physical infrastructure
  • C. Patching Amazon EC2 instances
  • D. Patching the network infrastructure

Correct Answer: C. Patching Amazon EC2 instances

The customer is responsible for security in the cloud, which includes the OS, networking, and firewall configuration.

ROUND 2

The Audit

Escalated difficulty. Focus on Automation and the CIA Triad.

Challenge: Round 2.1

The "Vault" Scenario

A bank (Tenant) wants to ensure that even if a Landlord employee walked into the server room, they couldn't read the files. Which security principle is applied?

Answer: Confidentiality (via Encryption)

Encryption keeps your "valuables" private even if the room is physically breached.

Challenge: Round 2.2

The "Emergency Sprinkler"

A Tenant wants their apartment to automatically add more space if guests arrive, keeping security identical. Which service automates this "reproducible" setup?

Answer: AWS CloudFormation

It acts as the architectural blueprint for your environment.

Challenge: Round 2.3

The "Building Inspector"

Which service continuously "walks the halls" to check if any Tenant has accidentally left their "door" (S3 Bucket) unlocked to the public?

Answer: AWS Config or Amazon Macie

They audit configurations and sensitive data discovery automatically.

ROUND 3

The "Toughy" Final

Multi-part analysis. Head-to-Head-to-Head.

The Managed Service Mystery

A Tenant switches from EC2 (Oracle) to RDS (Managed Service).

Part A: Who patched software on EC2?

Part B: Who patches it now on RDS?

Part C: If the Tenant leaks the Admin "Valve" (Credentials), who is at fault?

A: The Tenant (Customer)

B: The Landlord (AWS)

C: The Tenant (Always responsible for credentials)
